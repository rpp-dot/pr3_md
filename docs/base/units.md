# Юниты и их развитие

## Меню найма
Нажав на здание можно открыть меню найма и улучшения.

## Прокачка юнитов
Нажав на юнита можно открыть меню его улучшения.

## Эволюция
Юнит может эволюционировать при достижении максимального уровня.

## Список юнитов

| Юнит       | Роль         | Получение                     |
|-----------|--------------|-------------------------------|
| Рыцарь | ближний бой  | базовый юнит |
| Клоун | ближний бой  | эволюция из рыцаря |
| Странник | ближний бой | эволюция из рыцаря |
| Ведьма | дальний бой | базовый юнит |
| Ведьма+ | дальний бой | эволюция из ведьмы |
| Призрак | дальний бой | эволюция из ведьмы |
| Ангел | помощь | базовый юнит |
| Облако | помощь | эволюция из ангела |
| Инквизитор | дальний бой | эволюция из ангела |
