# Юниты и их развитие

## Меню найма

![](./images/recruiting.jpg)

Нажав на здание, можно открыть меню:
- улучшения здания (количество ячеек),
- найма новых юнитов.

---

## Прокачка юнитов

![](./images/levelup.jpg)

Нажав на юнита, вы открываете его меню улучшений.

---

## Эволюция

![](./images/evolve.jpg)
![](./images/evolved.jpg)

При достижении максимального уровня юнит может эволюционировать.

---

## Список юнитов

| Юнит       | Роль         | Получение                     |
|-----------|--------------|-------------------------------|
| Рыцарь | ближний бой  | базовый юнит |
| Клоун | ближний бой  | эволюция из рыцаря |
| Странник | ближний бой | эволюция из рыцаря |
| Ведьма | дальний бой | базовый юнит |
| Ведьма+ | дальний бой | эволюция из ведьмы |
| Призрак | дальний бой | эволюция из ведьмы |
| Ангел | помощь | базовый юнит |
| Облако | помощь | эволюция из ангела |
| Инквизитор | дальний бой | эволюция из ангела |
